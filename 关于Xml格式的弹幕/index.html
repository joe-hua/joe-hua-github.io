关于B站弹幕XML文件中的参数意义

发表于 2020-05-19 更新于 2021-05-05 分类于 -其他

B站弹幕XML文件中的部分代码:

在这部分代码中的

中的0,d43eabee,427026669这3个参数所代表的含义.

p这个字段里面的内容：
0,1,25,16777215,1312863760,0,eff85771,42759017
中几个逗号分割的数据

参数一

弹幕出现的时间 以秒数为单位。

参数二

弹幕的模式1..3 滚动弹幕 4底端弹幕 5顶端弹幕 6.逆向弹幕 7精准定位 8高级弹幕

参数三

字号， 12非常小,16特小,18小,25中,36大,45很大,64特别大

参数四

字体的颜色 以HTML颜色的十位数为准

参数五

Unix格式的时间戳。基准时间为 1970-1-1 08:00:00

参数六

弹幕池 0普通池 1字幕池 2特殊池 【目前特殊池为高级弹幕专用】

参数七

是发送者的ID，用于“屏蔽此弹幕的发送者”功能

参数八

是弹幕在弹幕数据库中rowID 用于“历史弹幕”功能。

d43eabee是用户代码
